¿Qué es Resident Security?

Es un sistema de seguridad para garantizar el ingreso de residentes y visitantes de un modo más seguro a unidades residenciales y evitar posible actos de delincuencia. Este sistema consiste en dos filtros: un sensor de huella y un sensor lector de tarjetas, se les otorgara tarjetas de acceso a la unidad residencial a cada residente y se registraran las huellas dactilares de cada uno para acceder a la zona o edifico de residencia.
